addUpTo() 這個function有 input n，n為一個大於0的正整數。 addUpTo() 要 return 1 + 2 + 3 + ..... + n 的值。 例如: addUpTo(5) = 1 + 2 + 3 + 4+ 5 = 15 addUpTo(100) = 1 + 2 + 3 + 4 + 5 + ....... + 100 = 5050 你可用等差數列公式來解，或是我建議你，用for loop來練習演算法! addUpTo(5); // 15 addUpTo(100); // 5050 addUpTo(5000); // 12502500 addUpTo(100000); // 5000050000
